      Out: Wednesday, Nov 14, 2018
      Due: 8pm Thursday, Nov 29, 2018

      You should complete this assignment in an IPython Notebook, using Python 2.7.

      A. The assignment

      You should download and launch this IPython notebook on your machine. You should read the problem statement and then solve Problem 1 from the theoretical part of hw4, as instructed on the assignment. The assignment contains clear instructions on what you need to fill in. Please do not change any function names or delete any cells!

      B. Your submission


      Once you have completed the assignment, you should save your notebook as <HW4solution.ipynb> and then do the following.
      1. Place the file <HW4solution.ipynb> in your home directory.
      2. Open your Terminal (in Mac OSX or Linux) and run the following commands in order. Replace <myUNI> below with your UNI.
        • cd
        • ls (should list HW4solution.ipynb as one of the files in the directory)
        • mkdir <myUNI_HW4>
        • cp HW4solution.ipynb <myUNI_HW4>/
        • tar -zcvf <myUNI_HW4.tgz> <myUNI_HW4>/
      3. You should now have the file <myUNI_HW4>.tgz in your home directory. Upload this file on Courseworks (under Assignments, hw4p).
        • For example, if your UNI is ab1234, you should upload the file ab1234_HW4.tgz (N.B.: your filename should not contain any brackets!).
